# Almods Electronics Website

A React-based website for Almods Electronics, featuring product recommendations, AI Matchmaker, and more.

## Structure
- `src/api/geminiService.js`: API logic
- `src/components/layout`: Header and Footer
- `src/components/common`: SectionTitle, FeatureCard, ProductCard
- `src/pages`: Home, Products, Contact
- `src/router/PageRouter.jsx`: Main router
- `src/utils/constants.js`: App constants
- `src/App.jsx`: Main container

## Getting Started
1. Install dependencies: `npm install`
2. Start the app: `npm start`

## Features
- AI-powered product recommendations
- Modern responsive design
- Contact form
